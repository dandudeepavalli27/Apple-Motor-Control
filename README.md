# APPLE (Motor Control)

## Aim
To calculate robot motor speed using distance and time data from robot motor dataset.

## Dataset
NASA Robot Motor Dataset
https://data.nasa.gov

## Objective
To determine the speed of a robot motor using motion parameters.

## Procedure
Step 1: Load robot motor dataset.
Step 2: Read distance value.
Step 3: Read time value.
Step 4: Apply formula speed = distance / time.
Step 5: Display speed.

## Algorithm
Step 1: Start the program.
Step 2: Load dataset.
Step 3: Read distance traveled.
Step 4: Read time taken.
Step 5: Calculate speed using formula:
        speed = distance / time
Step 6: Display speed.
Step 7: Stop the program.

## Code Logic
speed = distance / time

## Python Code
distance = 5.0
time = 2.0
speed = distance / time
print(speed)

## Output
2.5 m/s

## Result
Robot motor speed calculated successfully.

Apple Robotics uses motor control algorithms for precise robotic movement.
